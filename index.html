About Me

I’m a developer passionate about using technology to tell stories and solve problems. My favorite uses of tech lies in how technology can be used to express one’s creativity.
When I was younger, my parents placed me into a class for robotics, these classes were something I enjoyed, using simple robotics concepts to solve challenges. Then Covid-19 hit, I couldn’t go to these classes anymore since they were in person, that led me to taking classes in software development. These beginner level coding classes led me to taking a course for Unity, where I learned to make my own games. This is what triggered my love for software development. Since then, I’ve taken a Python course and an AI developer course. Other than taking courses, I’ve pursued learning on my own, like making my own old internet style website using html.
